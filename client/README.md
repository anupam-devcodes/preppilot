# PrepPilot Client

Frontend for **PrepPilot**, an AI-powered interview preparation platform.

Built with:

- React
- Vite
- React Router
- Axios
- Tailwind CSS

## Features

- Landing page
- Authentication UI
- Register with optional profile picture
- Login/logout flow
- Email verification page
- Forgot/reset password pages
- Protected dashboard route
- Session restore using httpOnly cookie and `/auth/me`
